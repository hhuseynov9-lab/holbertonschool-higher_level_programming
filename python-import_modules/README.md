# Python - Import & Modules: Task 0

Bu layihə Python-da modulların yaradılması, funksiyaların `import` edilməsi və `__name__ == "__main__"` məntiqini tətbiq etmək üçün kiçik bir tapşırıqdır.

## Tapşırıq Təsviri
`add_0.py` faylında yerləşən toplama funksiyasını əsas proqrama gətirmək və nəticəni formatlanmış şəkildə çap etmək.

### Fayl Strukturu
* `add_0.py`: `add(a, b)` funksiyasının olduğu modul.
* `0-add.py`: Əsas icra olunan skript.

### İstifadə Qaydası
Proqramı işə salmaq üçün terminalda aşağıdakı komandanı yazın:

```bash
python3 0-add.py
